Uso del Bot

Abyssus Bot es una herramienta diseñada para mejorar la experiencia en servidores de Discord. Al utilizar el bot, aceptas estos términos y te comprometes a usarlo de forma responsable.

Uso del Dashboard y Datos

Al hacer uso del Panel de Control (Dashboard) de Abyssus, el usuario reconoce y acepta lo siguiente:

  • OAuth2: El inicio de sesión se realiza a través de Discord. Abyssus no conoce ni almacena tu contraseña.
  • Gestión de Servidores: El bot accede a la lista de servidores donde posees permisos de administrador para permitir la configuración remota.
  • Sincronización: Los cambios realizados en el Dashboard (YouTube, Welcome, etc.) se guardan en tiempo real en nuestra base de datos sincronizada.
  • Tokens de Sesión: Se generan tokens temporales para mantener tu sesión segura. Estos expiran automáticamente tras un periodo de inactividad.

Edad Mínima

El uso del bot está destinado a usuarios que cumplen con la edad mínima requerida por Discord (13 años o más).

Responsabilidad del Usuario

  • El uso del bot es responsabilidad del administrador del servidor.
  • No debe utilizarse para actividades abusivas o ilegales.
  • El mal uso puede resultar en la restricción del servicio.

Restricciones de Uso

  • No utilizar el bot para spam o abuso.
  • No intentar explotar errores o vulnerabilidades.
  • No interferir con el funcionamiento del sistema.
  • No utilizar el bot con fines ilegales.
