Uso del Bot

Abyssus Bot es una herramienta diseñada para mejorar la experiencia en servidores de Discord. Al utilizar el bot, aceptas estos términos y te comprometes a usarlo de forma responsable.

Uso del Dashboard y Datos

Al hacer uso del Panel de Control (Dashboard), el usuario reconoce y acepta lo siguiente:

  • OAuth2: El inicio de sesión se realiza a través de Discord. Abyssus no conoce ni almacena tu contraseña.
  • Gestión de Servidores: El bot accede a la lista de servidores donde posees permisos para configuración remota.
  • Tokens de Sesión: Se generan tokens temporales y cifrados para mantener tu sesión segura.
  • Identidad: Solo se accede a tu ID y avatar de Discord para personalizar tu perfil dentro del panel.

Disponibilidad

Trabajamos para mantener una alta disponibilidad mediante infraestructura distribuida. No obstante, el servicio puede experimentar interrupciones por mantenimiento. No se garantiza el funcionamiento ininterrumpido.

Restricciones

  • No utilizar el bot para spam, abuso o acoso.
  • Prohibido explotar vulnerabilidades. Los errores deben reportarse al equipo de soporte.
  • No interferir con el funcionamiento del sistema o infraestructura de clústeres.
  • El mal uso resultará en la restricción inmediata del servicio.

Funciones Premium

Las funciones premium son opcionales. Aquellas obtenidas mediante recompensas o invitaciones no tienen valor monetario. Los pagos voluntarios no garantizan devoluciones salvo obligación legal.

Responsabilidad

El desarrollador no se hace responsable de daños o conflictos derivados del uso del bot. El uso es bajo tu propio riesgo y responsabilidad.
