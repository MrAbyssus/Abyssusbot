Uso del Bot

Abyssus Bot es una herramienta diseñada para mejorar la experiencia en servidores de Discord. Al utilizar el bot, aceptas estos términos y te comprometes a usarlo de forma responsable.

Uso del Dashboard y Datos

Al hacer uso del Panel de Control (Dashboard), el usuario reconoce y acepta lo siguiente:

  • OAuth2: El inicio de sesión se realiza a través de Discord. Abyssus no conoce ni almacena tu contraseña.
  • Gestión de Servidores: El bot accede a la lista de servidores donde posees permisos para permitir la configuración remota.
  • Tokens de Sesión: Se generan tokens temporales y cifrados para mantener tu sesión segura. Estos expiran automáticamente tras un periodo de inactividad.
  • Identidad: Solo se accede a tu ID y avatar de Discord para personalizar tu perfil dentro del panel.

Disponibilidad del Servicio

Trabajamos para mantener una alta disponibilidad mediante infraestructura distribuida (clústeres). No obstante, el servicio puede experimentar interrupciones por mantenimiento o fallos técnicos ajenos. No se garantiza el funcionamiento ininterrumpido.

Restricciones y Seguridad

  • No utilizar el bot para spam, abuso o acoso.
  • Prohibido explotar vulnerabilidades. Cualquier bug debe ser reportado inmediatamente al equipo de soporte.
  • No interferir con el funcionamiento del sistema o la infraestructura de clústeres.
  • El mal uso resultará en la restricción o suspensión del servicio.

Funciones Premium y Pagos

Las funciones premium son opcionales. Aquellas obtenidas mediante sistemas de recompensas, invitaciones o soporte no tienen valor monetario ni son reembolsables. Los pagos voluntarios no garantizan devoluciones salvo que la ley aplicable lo requiera.

Limitación de Responsabilidad

El desarrollador no se hace responsable de daños, pérdidas o conflictos derivados del uso del bot. El uso es bajo tu propio riesgo y responsabilidad.
